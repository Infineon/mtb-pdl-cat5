##################### Memory Layouts Supported By 55900A0 #######################
# This section documents the information about the layouts supported by 55900A0.
# Application can select any of the following supported layouts using
# MEMORY_LAYOUT_SEL variable either in bsp.mk or application make file.
# These Layouts are valid only for flash builds.
#
#   +-------------------+-----------------------------+
#   |  Layout Name      |  Description                |
#   +-------------------+-----------------------------+
#   |  f8_p8 (default)  |  8MB Flash, 8MB PSRAM       |
#   |  f8_p2            |  8MB Flash, 2MB PSRAM       |
#   +-------------------+-----------------------------+
#
